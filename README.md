Funcionamiento:
-El programa funciona con 5 funciones en el javascript, que hace leer la cantidad de movimientos que se necesite en un arreglo global, luego se puede eliminar algun movimiento por el nombre, despues se hace el resumen de los movimientos y el total de monto, posterior a eso se calcula el monto de ingresos y egresos para finalizar en el ingreso y egreso mas alto

Funciones creadas:
-registrarMovimiento()
-eliminarMovimiento()
-resumenGeneral()
-resumenTipo()
-masAltos()

Reflexion:
-Gracias a las herramientas de control de flujo facilitaron el obtener una respuesta segun el caso que aparezca, como puede ser al pedir los valores si no es el correcto dara una alerta y dara opcion a ingresar denuevo el valor, o para eliminar ya que es algo opcional, si no se requiere eliminar ningun movimiento solo se salta todo el funcionamiento de la funcion y listo
